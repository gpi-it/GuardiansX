# GuardiansX

# Installation

git clone https://github.com/gpi-it/GuardiansX.git

cd GuardiansX

copy app/js/config-sample.js app/js/config.js
// and insert your UrtheCast api keys in config.js

npm install

node server.js

browser: open http://localhost:3000
